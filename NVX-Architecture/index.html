— How to use this system —
01
📋
Add Your Devices
Edit devices.json with your NVX IP addresses, names, and locations. One JSON block per device.
"ip": "10.1.20.14"
"name": "Boardroom"
"location": "Floor 1"
02
⚙️
Tune Settings
Edit settings.json to change refresh rate, timeout, and how many devices poll at once.
"refresh_interval": 10
"fetch_timeout": 8
"max_concurrent": 8
03
▶️
Start the Server
Run the app. Browser opens automatically. Server fetches previews from all NVX devices.
python app.py
→ http://localhost:8000
04
🖥️
Monitor Dashboard
Open browser. See all NVX devices in a grid. Green = online. Red = offline. Click to expand.
http://localhost:8000
Filter by location tab
05
🔄
Hot Reload Devices
Edit devices.json and reload without restarting the server. Changes apply instantly.
localhost:8000/api/
devices/reload
06
📦
Share as EXE
Build a standalone exe. No Python needed on the target machine. Just unzip and run.
python build_exe.py
→ dist/NVX_Dashboard/
